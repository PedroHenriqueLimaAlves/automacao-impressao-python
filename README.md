# automacao-impressao-python

# 🖨️ Automação de Impressão

## 📌 Descrição
Script em Python desenvolvido para automatizar a separação de itens em pedidos de venda, diferenciando produtos gerais e cabos a partir de uma lista única de separação.

## 🚀 Tecnologias
- Python

## ⚙️ Funcionalidades
- Leitura e análise de arquivos de impressão de pedidos  
- Identificação automática de itens do tipo “cabos”  
- Separação dos itens em dois fluxos distintos  
- Geração de impressões independentes para produtos gerais e cabos  

## 💡 Contexto
Projeto criado para resolver um problema real no processo de separação de pedidos, onde todos os itens eram impressos juntos.  

Com a automação, pedidos contendo cabos passaram a ser divididos em duas impressões: uma geral e outra específica para cabos, direcionada a uma impressora dedicada no setor responsável.  

Isso permitiu que o responsável pela separação de cabos trabalhasse diretamente com as informações relevantes, aumentando a eficiência operacional e reduzindo erros no processo.
